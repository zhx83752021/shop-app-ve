# 零售电商 App - Vue 3 + TypeScript 启动说明

## 项目已完成内容

### ✅ 配置文件

- `package.json` - 项目依赖和脚本配置
- `vite.config.ts` - Vite 构建工具配置
- `tsconfig.json` - TypeScript 配置
- `tailwind.config.js` - TailwindCSS 样式配置
- `postcss.config.js` - PostCSS 配置

### ✅ 核心文件

- `src/main.ts` - 应用入口文件
- `src/App.vue` - 根组件
- `src/router/index.ts` - 路由配置
- `src/style.css` - 全局样式
- `src/types/index.ts` - TypeScript 类型定义

### ✅ 页面组件

- `src/views/Layout.vue` - 主布局（含底部导航栏）
- `src/views/HomePage.vue` - 首页（搜索、轮播、金刚区、秒杀、商品推荐）
- `src/views/DiscoverPage.vue` - 发现页（内容流、帖子列表、互动功能）
- `src/views/CartPage.vue` - 购物车（商品管理、优惠券、结算）
- `src/views/ProfilePage.vue` - 个人中心（用户信息、订单、会员、资产）

### ✅ 公共组件

- `src/components/ImageWithFallback.vue` - 图片容错组件

## 快速启动

### 1. 安装依赖

```bash
cd e:\app3\frontend
npm install
```

### 2. 启动开发服务器

```bash
npm run dev
```

开发服务器将在 http://localhost:3000 启动并自动打开浏览器。

### 3. 构建生产版本

```bash
npm run build
```

### 4. 预览生产版本

```bash
npm run preview
```

## 注意事项

1. **TypeScript 警告**: 在安装依赖前，IDE 会显示模块找不到的警告，这是正常的。运行 `npm install` 后会自动解决。

2. **图片加载**: 项目使用 Unsplash 的示例图片，如果网络较慢可能需要等待加载。

3. **移动端适配**: 项目使用 `max-w-md mx-auto` 限制最大宽度，在桌面浏览器中会居中显示。建议使用浏览器的移动设备模拟器查看效果。

4. **样式系统**: 使用 TailwindCSS，所有样式都通过工具类实现，保持了与原 Figma 设计的一致性。

## 开发提示

- 所有组件都使用 TypeScript 和 Composition API
- 样式完全使用 TailwindCSS 工具类
- 响应式数据使用 `ref` 和 `computed`
- 路由使用懒加载优化首屏性能
- 图片使用容错处理机制
